# Project Title

HireVision AI: An AI-Powered Mock Interview System with Emotion Detection and Resume Analysis

# Project Statement

HireVision AI is an AI-powered mock interview system that helps students and job seekers prepare for interviews. The system analyzes resumes, generates interview questions, detects emotions during interviews, and provides feedback to improve interview performance and confidence.

# Project Objectives

- Analyze the user's resume automatically.
- Generate interview questions based on the resume.
- Conduct AI-based mock interviews.
- Detect user emotions during interviews.
- Evaluate interview performance.
- Provide personalized feedback and suggestions.
- Improve interview skills and confidence.

# Module List

1. User Registration and Login Module
2. Resume Upload and Analysis Module
3. Interview Question Generation Module
4. Mock Interview Module
5. Emotion Detection Module
6. Performance Evaluation Module
7. Feedback and Report Generation Module
8. Admin Management Module
